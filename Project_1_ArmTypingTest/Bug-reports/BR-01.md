## Core Information
*   **Bug Type:** Functional
*   **Build Version:** 1.0
*   **Environment:** Windows 11 home 24H2 / Python 3.13.1 / pygame  2.6.1
## Description
Holding a key in the text field adds only one character. 
## Preconditions
1. The game has started. (The user pressed any key.)
## Steps To Reproduce (STR)
1. Press and hold a letter key. (e.g., 'A').
## Expected Result (ER)
*   The character repeats in the input field for the duration of the keypress.
## Actual Result (AR)
*   Character is added only once.
