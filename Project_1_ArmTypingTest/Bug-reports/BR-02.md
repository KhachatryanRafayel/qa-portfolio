**Status**: New\
**Severity:** Minor
## Core Information
*   **Bug Type:** Functional
*   **Build Version:** 1.0
*   **Environment:** Windows 11 home 24H2 / Python 3.13.1 / pygame  2.6.1
## Description
Holding a backspace key in the text field deletes only one character.
## Preconditions
1. The game has started. (The user pressed any key.)
## Steps To Reproduce (STR)
1. Type a few words or characters in the input field
2. Press and hold a backspace key.
## Expected Result (ER)
*   The characters in the input field removes for the duration of the keypress.
## Actual Result (AR)
*   Character is deleted only once.
